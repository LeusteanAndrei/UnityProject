# UnityProject
Raportul de recenzie am dori sa il primim pe Teams.
### Descriere Joc
Jocul se deschide normal si intra pe nivelul sample scene unde utilizatorul va putea testa sistemul de movement.
### Mechanics
w,a,s,d - moving
spacebar - jumping ( can double jump )
         - for the first jump, upon holding the jump button longer, the character jumps higher
shift - for sprinting on ground and if the character is in the air he will perform a quick dash
